

To get started:

1. Clone the repository:



   ```bash
https://github.com/jones345/accidents-bw.git
  bw Accidents dahsboard
   ```

2. Install the dependencies:

   ```bash
   # Using npm
   npm install

   # Using Yarn
   yarn
   ```

3. Start the development server:

   ```bash
   # Using npm
   npm run serve

   # Using Yarn
   yarn serve
   ```
